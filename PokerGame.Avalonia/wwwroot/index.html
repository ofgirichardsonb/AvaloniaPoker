Web UI Demonstration

This is a placeholder UI for the Avalonia-based Texas Hold'em Poker Game. In the future implementation, this will be replaced by a fully functional WebAssembly version of the Avalonia UI.

The full project architecture includes:

  • Cross-platform compatibility (Windows, macOS, Linux)
  • Microservices architecture with NetMQ messaging
  • Shared core business logic across platforms
  • REST API gateway for web clients
